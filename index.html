Binary Market Strategies

High-frequency inefficiency exploitation for Polymarket & Kalshi.

Project Summary

This engine identifies pricing inefficiencies in binary markets, specifically targeting the "Buy No Early" strategy. It monitors new market listings on Polymarket and Kalshi to detect hype-driven overpricing of "Yes" outcomes, signaling entry points for high-probability mean reversion trades.

Quick Start

1. Clone the repository:

git clone https://github.com/punyamodi/binary_market_engine.git

2. Install dependencies:

pip install requests

3. Run the demo:

python run_demo.py
